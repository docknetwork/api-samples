# Dock API JS examples
Javascript examples for the Dock API

Create a `.env` file with the following values:

    API_URL=http://localhost:8000 
    API_KEY=<INSERT YOUR KEY HERE>


Run a workflow:
    `npm run dids`
    `npm run schemas`
    `npm run credentials`
    `npm run registries`
    `npm run anchors`
    `npm run presentations`

Run all workflows:
    `npm run all`
